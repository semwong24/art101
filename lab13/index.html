Lab 13: Loops

Challenge

The challenge of this lab was to create a FizzBuzz program that prints numbers from 1 to 200. If the number is divisible by 3, 5, or 7, it prints "Fizz", "Buzz", or "Boom", respectively. If divisible by more than one factor, it combines them (ex. "FizzBuzz").

Problems

One issue I ran into was ensuring the program printed the correct combinations (ex. "FizzBuzz" for 15). I solved this by checking for combinations first. The output was initially a long list, so I used CSS Grid to display the results in columns, making it more readable.

Reflection

This lab reinforced my understanding of loops, conditionals, and DOM manipulation. The main challenge was getting the combination logic right and making the output visually organized. It was a useful exercise, and I am happy with the results.

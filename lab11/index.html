Lab 11: JavaScript Events and Forms

Challenge

This lab's goal was to use JavaScript and jQuery to respond to user input. On the page, I built a form where a user can enter their name, click a button, and see the letters sorted alphabetically using a custom function.

Problems

A problem that came up was making sure the jQuery selectors matched the HTML elemnt IDs. Using DEFER helped ensure the script didn't run before the page loaded. Styling and layout also took some trial and error to get clean results.

Reflection

This was a fun and manageable lab. It helped solidify how events and DOM updates work. The interactivity felt rewarding, and testing with the console made debugging easier. I think I would like to explore adding more features next time.

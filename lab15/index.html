Lab 15: AJAX

Challenge

The challenge of this lab was to use jQuery and AJAX to retrieve data from an external API and display it on a webpage in response to user interaction. We needed to test an endopint, fetch JSON data, and format it for display, while keeping everything clean and styled.

Problems

One issue we ran into was understanding the API response structure from the Pok√©API, especially how to access nested objects like abilities and sprite URLS. We solved this by logging the data to the console and checking how the properties were organized. Another small issue was making the page look good visually, but we fixed that by adjusting CSS and hiding the output until the botton is clicked.

Reflection

This lab was a good introduction to working with APIs and asynchronous JavaScript. It was satisfying to fetch live data and display it dynamically on the page. We also got to practice debugging using the console and refining our CSS to create a better user experience. Overall, it helped reinforce how frontend and backend communication works on the web.

Lab 12: Conditionals

Challenge

The challenge was to use JavaScript conditionals to create a "Sorting Hat" experience that reacts to user input and outputs a result based on logic.

Problems

I didn't run into any major problems while working on this lab. The most would be that at first I wasn't sure how to style the output dynamically with jQuery, but testing in the console helped reaffirm that the HTML and DOM can be updated together.

Reflection

This lab helped reinforce how conditionals work in JavaScript and gave me creative control over the user experience. It was fun and satisfying to test live changes.

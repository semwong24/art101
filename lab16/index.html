Lab 16: JSON and APIs

Challenge

The challenge of this lab was to learn how to work with APIs and JSON data using jQuery. We practiced accessing external data from the XKCD API and displaying specific elements like the comic image, title, and alt text on our webpage.

Problems

At first, we ran into a CORS (Cross-Origin Resource Sharing) error when trying to access the XKCD API directly from our local environment. This was solved by using a temporary access proxy from cors-anywhere.herokuapp.com, which allowed us to retrieve and display the comic data correctly. We also used console.log to explore the structure of the JSON object and ensure everything was working properly.

Reflection

This lab helped us understand how to handle real-world API data and dynamically insert content into a webpage. It felt rewarding to see the comic load successfully after troubleshooting the CORS issue. We put consistent effort into testing the endpoint, understanding the JSON structure, and debugging the AJAX request. Overall, this was a solid intro to working with APIs.

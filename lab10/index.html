Lab 10 - JavaScript for the Web

Challenge

The challenge of this lab was to use JavaScript and jQuery to modify the DOM by generating and appending new elements when a user interacts with a button. This lab explores how JavaScript can be used to make websites more interactive and responsive.

Problems

Initially, I had trouble getting the JavaScript to run properly. The event listener didn't seem to work at first, but I realized that I had placed the script code before the DOM had fully loaded. Adding the DEFER attribute to the script tag fixed the issue. I also had to move the button outside of the #output div to avoid accidentally overwriting it with new content.

Results

The final result is a functional web page where clicking the "Say Something" button generates a new chat-style text bubble with a random snippet of text. I used jQuery to handle the button click and DOM manipulation, and the CSS to style the bubbles. The output successfully simulates a chat interface and demonstrates dynamic interaction using JavaScript and jQuery.

Bonus Tasks

As an extra feature, I added a message alignment system that alternates each bubble left and right, simulating a real converstation. I also included an input box so the user can type their own message. If the box is left empty, the system defaults to a randomly generated message. Pressing the "Send" button or hitting Enter submits the message and appends it to the chat thread.

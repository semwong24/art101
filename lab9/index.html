Lab 9 - Libraries & jQuery

Challenge

The challenge of this lab was to practice using the jQuery library to dynamically manipulate HTML elements. Specifically, we created buttons that could toggle special styling on different sections of the page.

Problems

One issue I ran into was that the buttons at first weren't showing up the the correct sections. I realized this was because of how I used jQuery's 'append()', where the buttons were being added inside child elements instead of directly inside the right section. Once I adjusted the selectors and structure, it worked out correctly.

Results

The page now successfully shows three buttons-one under each section. When clicked, each button toggles a custom '.special' style for that section, changing background color, border, and text styling using jQuery.
